## Пайплайны (проверенные, повторяемые)

### 1. Обработка фото лота → чистый белый фон
Полный алгоритм и КОРОТКИЙ УНИВЕРСАЛЬНЫЙ ПРОМПТ — в `CLAUDE.md` (главное правило).
Кратко: nano banana pro 2k (Higgsfield MCP) с промптом без имён → при отказе
ретрай 1–2 → Gemini API напрямую → жёсткий блок = cv2 perspective-crop реальных
пикселей. `remove_background` — только для объёмных предметов.
После генерации — ДВА круга визуальной проверки (Read картинки): автограф не
изменён/не дорисован (был случай — nano ДОРИСОВАЛ подпись на обложке книги;
ловится только глазами), текст не выдуман, фон чистый.
eBay-фото качать в `s-l1600`. Higgsfield workflow: media_upload → PUT → media_confirm
→ generate_image(medias=[{role:image,value:media_id}]) → job_display → скачать rawUrl.

### 2. Рендер дека → PDF
```
/opt/pw-browsers/chromium --headless --no-sandbox --disable-gpu \
  --virtual-time-budget=30000 --print-to-pdf=OUT.pdf --no-pdf-header-footer deck.html
```
Print CSS уже внутри деков (@page 1280x720 или 720x1280). Постраничная проверка:
`pdftoppm -r 92 -jpeg` → монтаж → смотреть глазами.

### 3. Самодостаточный веб-артефакт (шаринг клиенту)
Python: заменить `<link Google Fonts>` на инлайн `17_stargift/fonts/inline.css`,
все `src="img*/..."` → base64 JPEG (thumbnail ≤1200px, q80-82, лимит файла 16 МБ),
добавить @media screen блок (стек слайдов + transform-scale под ширину окна).
Образец сборщика — в истории git (коммиты «artifact») или собрать заново по описанию.
