# Интейк экспоната Relictum — «дали продукт → разместили за 15 минут»

Единый источник данных: **`shared/catalog.js`**. Добавил объект туда — он появился
в каталоге (с фильтрами и сортировкой), получил свою карточку `object.html?id=R–XXXX`,
попал в «сопутствующие». Больше НИГДЕ дублировать не нужно.

## Шаг 1 · Фото (стандарт дома)
- Основной кадр: объект на ivory seamless #F4F0E8, мягкий боковой свет, 4:5.
  Если фото нет — сгенерировать по промпт-формуле из `.claude/skills/relictum-product-promo/SKILL.md`.
- Файл: `shared/img/prod_<slug>.jpg` (JPEG ~1600px по длинной стороне, quality 80;
  оригинал 2K PNG — в `05_art/products/`).

## Шаг 2 · Запись в shared/catalog.js
Скопировать любой объект и заполнить поля:
```js
{
  "id": "R–0208",                    // шифр дома: R–<мир><номер>. Тире — ДЛИННОЕ (–)
  "slug": "spinosaurus-tooth",       // латиницей, для ссылок
  "world": "vita",                   // vita | terra | cosmos | grand | editions
  "worldLabel": "Жизнь",
  "category": "Динозавры",           // тип объекта (фильтр)
  "period": "Мел",                   // эпоха (фильтр): Космос/Триас/Юра/Мел/Палеоген/Неоген/Девон/Плейстоцен
  "region": "Марокко",               // регион находки (фильтр)
  "priceValue": 920000,              // число для сортировки; null = «по запросу»
  "price": "920 000 ₽",              // как показываем; или "Цена по запросу"
  "name": "Зуб спинозавра",
  "latin": "Spinosaurus aegyptiacus",
  "meta": "Кем-Кем, Марокко · 93–112 млн лет<br>12 см · речной хищник",  // 2 строки карточки
  "img": "prod_spinosaurus_tooth",   // имя файла в shared/img без .jpg
  "age": "93–112 млн лет",           // для страницы объекта
  "location": "Плато Кем-Кем, Марокко",
  "size": "12 см по дуге",
  "description": "1–2 предложения истории в голосе дома: факт → образ. Без «купи» и восклицаний.",
  "href": ""                         // ОПЦИОНАЛЬНО: путь к промо-странице, если сделана
}
```
