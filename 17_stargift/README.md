# Stargift — презентации экспонатов

Пилот конвейера презентаций для компании **Stargift** (stargift.ru): из большой клиентской подборки
(PDF из Keynote) собирается сокращённая имиджевая дека в фирменном стиле — тёмная гамма сайта
stargift.ru, золото, серифы, рукописный логотип.

## Файлы

- `football_wc2026.html` — дека «Легенды мирового футбола · к ЧМ-2026», 19 слайдов, 12 экспонатов.
  Управление: `←` / `→` / пробел / клик; печать в PDF — `Ctrl+P` (слайды 1280×720);
  прямая ссылка на слайд — `#N` в адресе.
- `img/` — фото экспонатов и логотип, извлечённые из исходного PDF (pdfimages/pdftoppm),
  плюс реальные фотографии матчей с разделительных страниц той же подборки.

## Как обновлять (шаблон конвейера)

1. Прислать новый PDF подборки (или список позиций с сайта) и сказать, сколько позиций оставить.
2. Из PDF извлекаются: страницы-разделы, тексты описаний, цены, сертификация, фото экспонатов.
3. Отбираются топ-позиции (по цене/статусу, с покрытием всех разделов подборки).
4. Данные подставляются в блоки `<section class="slide lot">` — по одному на экспонат;
   разделы — блоки `<section class="slide divider">` с фото матча фоном.
5. Прогон скриншотов в headless Chromium для визуальной проверки.

Слайд экспоната принимает два типа фото: `ph dark` — тёмная предметная съёмка на весь блок,
`ph card` — светлое каталожное фото в паспарту цвета слоновой кости.
